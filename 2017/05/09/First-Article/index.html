Git

Git是什么

Git是一个分散式版本控制软件，是目前世界上最先进的分布式版本控制系统（没有之一）。

通俗说，就是帮助开发者更好的管理项目的版本，尤其适用团队开发。

Git的安装

Windows用户直接百度git下载git客户端，傻瓜式安装即可，安装完成后，打开命令行输入：git --version 如果看到git的版本，说明安装成功了。

如果返回命令没有找到的错误，请检查环境变量中是否有git的路径。

Mac用户，打开iterm或terminal直接输入： git --version，不出意外，会直接返回git版本，因为Mac其实自带了git与svn…偷笑一会~~

Git基础命令

git init 创建git版本库，这会在当前文件夹下生成 .git 文件，
如果没有那就使用ls -a查看所有文件，应该是被隐藏了。

没事不要修改.git文件，
生成了就不需要去管它了。

在连接远程库时你需要做的

  1. 注册GitHub账号（GitHub是提供git仓库托管服务的一个神奇的网站）

总结以及换个思路

我们总结一下git的使用：

  1. 安装git，这一步几乎不用多说，只要命令行中正确返回了git版本，基本就OK。

  2. 生成SSH Key

  3. 创建本地仓库，随便在某处新建文件夹，然后在文件夹内输入 git init 即可。生成了 .git 文件说明成功了。
    .git 文件可能是隐藏的，在命令行中使用 ls -a 可以看到它，Windows用户可以百度如何让电脑显示隐藏文件。

  4. 在本地仓库新建文件，并使用 git add filename 将该文件提交至暂存区，并使用 git commit -m "描述" 命令，将其加入版本库。

  5. 在GitHub上新建一个远程仓库demo，在本地仓库文件夹下使用 git remote add origin git@github.com:yourgithubusername/demo.git
    命令，将本地仓库与其关联。并使用 git push -u origin master 将本地仓库内容推送到远程仓库。第一次需要使用-u ，后面就不需要了
