百度编辑器的调用

发表于 2017-02-04 | 分类于 Web |

2017第一天上班

年前在写一个商城后台系统，上传商品时添加描述调用的是百度编辑器。由于之前公司后台的百度编辑器巨卡，这次我决定自己试试。

这是官网，大家可以先看看文档。

直接进入正题

这是HTML代码

<div class="control-group">
        <label class="control-label">商品描述</label>
        <div class="controls">
        <script type="text/plain" id="content" name="content"></script>
        </div>
 </div>

这是JS引入文件

<script type="text/javascript">
//编辑器路径定义
var editorURL = GV.WEB_ROOT;
</script>
<script type="text/javascript"     src="__PUBLIC__/js/ueditor/ueditor.config.js"></script>
<script type="text/javascript"     src="__PUBLIC__/js/ueditor/ueditor.all.min.js"></script>
<script type="text/javascript">
//实例化编辑器
//initialFrameHeight/Width自定义编辑器宽高
editorcontent = new baidu.editor.ui.Editor({});
editorcontent.render('content');
try {
    editorcontent.sync();
} catch (err) {

}
</script>

一定要在引入ueditor.config.js和ueditor.all.min.js之前定义编辑器路径，以及在引入之后实例化编辑器

自定义编辑器宽高

在查了资料之后发现，其实很简单。

editorcontent = new  baidu.editor.ui.Editor({initialFrameHeight:200,initialFrameWidth:600 });

只需要在Editor()中加入initialFrameHeight 和 initialFrameWidth 定义宽高，且无需加px。
